Wrinkle Wrumble

Unity / Game / NetCode

Description

The goal of this project was to create a humorous "party style" mutliplayer game. In Wrinkle Wrumble, you push or throw objects in a small retirement house to defeat your opponent.

In this project, the 3rd party PUN module was used extensively to give us the networking tools we needed to make this all possible. Currently the game is still in its alpha stage and there are no current plans to develop it any further. Instead, this project will mostly serve as a benchmark for what we can do (sans the polish and balance testing).

You can click the button below to launch the game and play with friends.
Technologies

Unity, C#, JavaScript, Mixamo, Fuse, HTML5, PUN (Photon Unity Networking)
