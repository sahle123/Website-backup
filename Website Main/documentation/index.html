Markup Markup customization and highlights

This theme is has a responsive layout with support for up to 12 columns. All of the main content of a section is found inside the section tag. This theme contains 4 html files, for Homepage, Portfolio, Project and Contact

Portfolio

Portfolio are found in two places in this template. First on the main index.html page, where it shows a few of the most recent items. And second is on the index.html page found inside the portfolio folder, where it displays all items.


Portfolio Filter Plugin

The filter plugin(MixItUp) is found on the main index.html page and portfolio.html page.

To the right is the general format of the plugin.

For more info click visit http://mixitup.io/


Filter Buttons

Each button (li) should must contain the class 'filter'. To determine what items a button filters, add the corresponding filter type into the attribute 'data-filter'. You may add mutiple filter type to combine the filtering.

<ul>
    <li class="filter" data-filter="dogs"><⁄li>
    <li class="filter" data-filter="cats"><⁄li>
    <li class="filter" data-filter="krakens"><⁄li>
    <li class="filter" data-filter="dogs cats"><⁄li>
<⁄ul>

Filter Grid

Each item (li) should must contain the class 'mix'. To determine what category an item belongs to, add the corresponding category type into the class. You may add mutiple filter type to combine the filtering.

Please note that the 'ul' must have an id attribute, in this scenario it is 'Grid'.

 <ul id="Grid">
    <li class="mix dogs"><⁄li>
    <li class="mix cats"><⁄li>
    <li class="mix krakens"><⁄li>
    <li class="mix dogs cats"><⁄li>
    <li ...
<⁄ul>

CSS

Add the following code to the main css file.

#Grid .mix{
    opacity: 0;
    display: none;
}

Activate Plugin

The code to the right is used to activate the plugin. It uses the id attribute from the 'ul' the let the plugin know what is is filtering.

 $(function(){ 
    $('#Grid').mixitup();
});
