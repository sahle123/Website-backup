Wrinkle Wrumble

Unity / Code / Game / NetCode

The goal of this project was to create a humorous "party style" mutliplayer game. In Wrinkle Wrumble, the goal is to push or throw objects in a small retirement house to defeat your opponent.

In this project, the 3rd party PUN module was used extensively to give us the powerful networking tools we needed to make this all possible. Currently right now the game is still in its alpha stage and there are no current plans to develop it any further. Rather, this project will mostly serve as a benchmark for what we can do (sans the polish and balance testing).

You can click the button below to launch the game and play with friends. If the link does not work, please let me know by going to the contact tab so I can fix the it immediately.

Technologies Used
Unity3D, C#, JavaScript, Maximo, Fuse, HTML5, PUN (Photon Unity Networking)
